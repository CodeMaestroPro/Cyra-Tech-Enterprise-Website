# Cyra-Tech Enterprise Platform

Built on Vision. Driven by Intelligence.

## Stack

- Laravel 12
- JavaScript + Blade
- Tailwind CSS 4
- MySQL

## Setup

```bash
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
composer install
npm install
npm run build
composer dev
```
